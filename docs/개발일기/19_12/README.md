  - _ : omit :	빼라
  - _.get(obj, ‘name’, ‘nkh’) : undified라면 ’nkh’를 가져온다
  - _.sortBy(arr, [‘created_at’,’name’])
  - _.pick(obj, ‘name’, ‘age’) - obj에서 name값만 가져온다.
  - _.omit(obj, ‘name’,’age’) - obj에서 name, age 빼고 가져온다
