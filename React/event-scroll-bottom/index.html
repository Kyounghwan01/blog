# 아래로 내리는 이벤트 추가

저는 댓글이 추가될 경우 이벤트가 발생하여, 스크롤을 가장 밑으로 내리는 작업을 하였습니다.

댓글이 추가될 경우 store의 editDone이 true로 바뀌게됩니다.
